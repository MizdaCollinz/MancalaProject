# MancalaProject
Mancala Implementation focusing on quickly creating a modifiable implementation

Mancala is an old game also known as Kalah which involves two players.
Each player has a set number of houses on their side of the board and one store at the right end of the board relative to them.


Solid Branch focusses on implementing the SOLID Object Oriented programming principles 
 - Dependency Inversion Principle
 - Interface Segregation Principle
